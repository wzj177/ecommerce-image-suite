# 电商平台规范参考

## 国内平台

### 淘宝 / 天猫
- **主图尺寸**：800×800px 或 1:1（最低 700px）
- **主图要求**：纯白底（RGB 255,255,255），商品占比 ≥ 80%
- **附图数量**：最多 20 张
- **格式**：JPG/PNG，单张 ≤ 3MB
- **文案风格**：中文，情感化，强调品质与性价比，突出折扣/促销
- **禁止**：牛皮癣广告、过度文字水印、假冒品牌标识
- **套图建议顺序**：白底主图 → 卖点图 × 2 → 材质图 → 场景图 → 模特图 → 多场景图

### 京东
- **主图尺寸**：800×800px（最低 400×400px）
- **主图要求**：白底，无水印，商品居中
- **附图数量**：最多 10 张
- **格式**：JPG，单张 ≤ 2MB
- **文案风格**：中文，专业理性，强调参数、品质、服务保障
- **特色**：可加入品质认证标识、正品标签

### 拼多多
- **主图尺寸**：750×750px（最低 480×480px）
- **主图要求**：白底或场景图均可，但白底转化率更高
- **附图数量**：最多 10 张
- **格式**：JPG/PNG，单张 ≤ 3MB
- **文案风格**：中文，简洁直白，强调价格优势、实用价值
- **特色**：价格促销信息可在图片中呈现，多买多省

### 抖音电商（抖店）
- **主图尺寸**：1:1（800×800px）或 3:4（600×800px）
- **短视频封面**：9:16（1080×1920px）
- **文案风格**：中文，年轻化，场景感强，与短视频内容联动
- **特色**：突出"种草"感，生活化场景优先，强调穿搭搭配

### 小红书
- **图片尺寸**：3:4（1080×1440px）或 1:1（1080×1080px）
- **文案风格**：中文，生活方式，情感共鸣，"博主推荐"风格
- **特色**：美感优先，生活化场景，可带手写风格文字

## 国际平台

### Amazon
- **主图尺寸**：最小 1000×1000px，推荐 2000×2000px（支持缩放）
- **主图要求**：
  - 纯白背景（RGB 255,255,255）
  - 商品占图片面积 ≥ 85%
  - 不可有文字、水印、边框、装饰
  - 不可有多个商品（变体除外）
- **附图数量**：最多 9 张
- **格式**：JPEG（首选）/PNG/TIFF，颜色模式 sRGB
- **文案风格**：英文，专业简洁，突出功能与规格，符合搜索习惯
- **A+ 内容图**：宽度 970px 或 1464px，高度不限

### Shopify / 独立站
- **推荐尺寸**：2000×2000px（1:1）或 2000×1500px（4:3）
- **风格自由度**：最高，可根据品牌调性定制
- **文案风格**：品牌化，英文或多语言，强调品牌故事与生活方式
- **Banner 图**：1920×600px 或 1440×500px

### eBay
- **主图尺寸**：最小 500×500px，推荐 1600×1600px
- **要求**：白底为主，商品清晰

## 文案语言规范

### 中文文案（国内平台）
```
主标题格式：[卖点关键词] + [商品类型]（≤ 12 字）
副标题格式：动词 + 效果描述（≤ 15 字）
示例：
  主标题：宽松透气，遮肉显瘦
  副标题：无束缚感，穿出好身材
```

### 英文文案（国际平台）
```
主标题格式：[Adjective] + [Product Type] 或 功能型标语（≤ 5 words）
副标题格式：Benefits-focused, active voice（≤ 8 words）
示例：
  主标题：LOOSE FIT DESIGN
  副标题1：Unrestricted Movement
  副标题2：Comfortable and Flattering
```

## 平台套图优先级建议

| 平台 | 必做图 | 推荐图 | 可选图 |
|------|--------|--------|--------|
| 淘宝/天猫 | 白底主图、卖点图 | 材质图、模特图 | 多场景图 |
| 京东 | 白底主图、核心卖点图 | 材质图、场景图 | 模特图 |
| 拼多多 | 白底主图 | 卖点图 | 场景图 |
| 抖音 | 场景展示图、模特图 | 卖点图 | 多场景图 |
| Amazon | 白底主图 | 核心卖点图、场景图 | 模特图、材质图 |
| 独立站 | 模特图、场景图 | 白底主图、材质图 | 多场景图 |
